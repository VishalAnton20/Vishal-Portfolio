Description

  • Built and evaluated multiple regression models, including Linear Regression, Random Forest, and XGBoost.
  • Achieved high prediction accuracy with XGBoost and Random Forest models, outperforming baseline models.
  • Conducted model evaluation using R², RMSE, MAE, and MSE metrics for performance comparison.
  • Visualized data distributions, correlations, and feature importance using Seaborn and Matplotlib.
  • Optimized model performance using hyperparameter tuning with GridSearchCV.
  • Delivered a predictive solution that can assist dealerships and individuals in estimating fair car resale values.
